# HOI4 Minecraft Website

This is a copy of the heartsofminecraft.com website with an added Blog page.

## Pages

- Home: index.html
- Shop: shop.html
- Live Map: map.html
- Statistics: stats.html
- Blog: blog.html (new page for user-submitted blogs/news)

## Blog Feature

The blog page allows users to submit their own blogs/news, which are stored locally in the browser.

To run the website, open index.html in a web browser.
